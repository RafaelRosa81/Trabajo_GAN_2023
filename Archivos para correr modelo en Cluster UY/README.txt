README

Se presentan algunos archivos para el caso en que se quieran ejecutar las corridas que se presentan en el informe. 
El nombre de cada archivo muestra de qué dataset se trata y de qué función de pérdida. 


Todos los archivos están armados para ser corridos desde el home del usuario en el Cluster UY. 
La carpeta "one-shot-synthesis" debe estar localizada en el home del usuario en el Cluster UY.

Para ver detalles de ejecución y creación del ambiente para ejecución del modelo (osmis), revisar: https://github.com/boschresearch/one-shot-synthesis
